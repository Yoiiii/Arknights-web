## 安装

运行项目需要安装node环境

运行项目  

```sh
npm install
npm run serve
```

## 项目目录

```sh
├── public                  // 公共文件
├── src                     // 资源文件
│   ├── assets              // 资源文件夹
│   ├── components          // 组件文件夹
│   ├── router              // 路由文件夹
│   ├── views               // 页面文件夹
├── App.vue                 // App组件
├── main.js                 // 入口文件
```
